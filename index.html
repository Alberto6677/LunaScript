LunaScript

Le premier langage français
pensé pour le Web moderne

LunaScript est un langage simple, clair et entièrement en français. Conçu pour apprendre plus vite, créer plus facilement, et rendre le code accessible à tous — débutants comme passionnés.

Il est open-source, rapide à intégrer, et entièrement pensé pour les développeurs francophones.

Premier script en LS 

def nom = "Alberto67"
msg(nom)

repeter 3 {
    msg("Salut")
}

si (1) {
    msg("Condition vraie")
}

Installer LunaScript

Ajoute ceci à ton HTML : 

<script src="https://lunascript.onrender.com/cdn/ls.js"></script>
<script type="ls">
msg("LunaScript chargé !")
</script>
